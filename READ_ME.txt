********** Requirements **********
 - OpenCV
 - Tensorflow
 - Tesseract
 - Scikit-learn
 - Numpy
 - Matplotlib
 - PyPDF
 - Imutils




********** Instructions **********

 - Install Flask 
 - Set the app to app_basic.py 
 - Run the app at the selected port
 - Select the desired mode of operation
 - To view the result after submission, go through one of the following:
	1. To run ImageOCR, take input image from ImageOCR/images 
	2. To run PDFOCR, take input from PDF folder
	3. To run SheetsOCR, take input from forms folder
